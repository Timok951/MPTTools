О сайте

MPT Tools

Система учёта оборудования, заявок, списаний и рабочих мест.

Используемые сервисы

  • Яндекс.Карты API используются для выбора локации рабочих мест.
  • Адрес и координаты сохраняются только для внутренних задач учёта.
  • Доступ к разделам контролируется ролевой моделью.

Правовая информация по картам

  • Условия использования отдельных сервисов Яндекс.Карт
  • Условия использования сервиса Яндекс.Карты

При использовании карт соблюдаются требования правообладателя сервиса.
