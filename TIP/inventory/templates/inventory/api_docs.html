/api/v1/

Поддерживаются две схемы авторизации: браузерная сессия и токен DRF. Для получения токена отправьте логин и пароль на /api/v1/auth/token/, для отзыва используйте /api/v1/auth/token/revoke/.

Роли применяются по типу ресурса: администратор имеет полный доступ, склад может изменять складские сущности и корректировки, а Sysadmin и Builder работают со своими заявками, таймерами и выдачами.
